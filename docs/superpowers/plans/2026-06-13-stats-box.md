# Box "Total Arrecadado" Implementation Plan

> **For agentic workers:** REQUIRED SUB-SKILL: Use superpowers:subagent-driven-development (recommended) or superpowers:executing-plans to implement this plan task-by-task. Steps use checkbox (`- [ ]`) syntax for tracking.

**Goal:** Add a box, visible to all logged-in participants, showing the total
amount raised in the bolão (number of unique e-mails in the participants
spreadsheet × R$ 20,00).

**Architecture:** Pure static HTML/CSS/JS, no build step, no test runner.
Add one new `<section>` to `index.html`, one CSS block to `style.css`, and
one JS function (`buildStatsBox`) to `app.js` that fetches and parses the
existing `PARTICIPANTS_CSV_URL` CSV (reusing `fetchCSV`/`parseCSV`), counts
unique e-mails, and renders the total.

**Tech Stack:** Vanilla JS, CSS custom properties, CSV fetched live from a
published Google Sheet.

**Spec:** `docs/superpowers/specs/2026-06-13-stats-box-design.md`

```js
// Conta e-mails únicos na planilha de participantes e exibe o total
// arrecadado (cada e-mail = 1 pessoa = R$ 20,00).
async function buildStatsBox() {
  const box = document.getElementById("stats-box");
  if (!box) return;

  box.innerHTML = `<div class="stats-loading">⏳ Calculando total arrecadado...</div>`;

  try {
    const text = await fetchCSV(PARTICIPANTS_CSV_URL);
    const rows = parseCSV(text);

    const emails = new Set();
    rows.forEach(row => {
      const email = (
        row["Endereço de e-mail"] ||
        row["Email"] ||
        row["email"] ||
        Object.values(row)[1] ||
        ""
      ).trim().toLowerCase();
      if (email) emails.add(email);
    });

    const total = emails.size * BET_VALUE_PER_PERSON;
    const perPerson = BET_VALUE_PER_PERSON.toLocaleString("pt-BR", { minimumFractionDigits: 2 });

    box.innerHTML = `
      <div class="stats-value">💰 R$ ${total.toLocaleString("pt-BR", { minimumFractionDigits: 2 })}</div>
      <div class="stats-detail">${emails.size} ${emails.size === 1 ? "participante" : "participantes"} · R$ ${perPerson} cada</div>
    `;
  } catch (err) {
    box.innerHTML = `<div class="stats-error">⚠️ Não foi possível calcular o total arrecadado.</div>`;
    console.error(err);
  }
}

// ---------- BETS TABLE ----------
```
